# Logo Assets

Transparent PNG logo assets for HTML usage.

## Files

- `assets/lotte-rental-logo-red.png`
- `assets/lotte-rental-logo-gray.png`
- `assets/mockups/mycar-save-guide-mockup.png`
- `assets/mockups/monthly-rental-guide-mockup.png`

## Raw URL

After pushing this folder to a public GitHub repository, use the raw URL format below.

```html
<img
  src="https://raw.githubusercontent.com/YOUR_GITHUB_USERNAME/lotte-rental-assets/main/assets/lotte-rental-logo-red.png"
  alt="LOTTE rental"
  width="240"
>
```

Gray version:

```html
<img
  src="https://raw.githubusercontent.com/YOUR_GITHUB_USERNAME/lotte-rental-assets/main/assets/lotte-rental-logo-gray.png"
  alt="LOTTE rental"
  width="240"
>
```

Mockup references:

```html
<img
  src="https://raw.githubusercontent.com/YOUR_GITHUB_USERNAME/lotte-rental-assets/main/assets/mockups/mycar-save-guide-mockup.png"
```

## jsDelivr CDN

For a cleaner CDN URL after the repository is public:

```html
<img
  src="https://cdn.jsdelivr.net/gh/YOUR_GITHUB_USERNAME/lotte-rental-assets@main/assets/lotte-rental-logo-red.png"
  alt="LOTTE rental"
  width="240"
>
```

## Publish

```bash
git init
git add .
git commit -m "Add LOTTE rental transparent logo assets"
git branch -M main
git remote add origin https://github.com/YOUR_GITHUB_USERNAME/lotte-rental-assets.git
git push -u origin main
```
